# typewriter-kotlin

> Kotlin emitter for the [typewriter](https://github.com/aarambh-darshan/typewriter) SDK.

[![Crates.io](https://img.shields.io/crates/v/typewriter-kotlin.svg)](https://crates.io/crates/typewriter-kotlin)
[![Docs.rs](https://docs.rs/typewriter-kotlin/badge.svg)](https://docs.rs/typewriter-kotlin)

## What It Generates

| Rust | Kotlin |
|---|---|
| `struct` | `@Serializable data class` |
| Simple `enum` | `@Serializable enum class` with `@SerialName` |
| Tagged `enum` | `@Serializable sealed class` with data class subtypes |
| `Option<T>` | `T? = null` |
| `Vec<T>` | `List<T>` |
| `HashMap<K,V>` | `Map<K, V>` |

## Example Output

```kotlin
// Code generated by typewriter v0.2.0. DO NOT EDIT.
// Source: UserProfile

package types

import kotlinx.serialization.Serializable
import kotlinx.serialization.SerialName

@Serializable
data class UserProfile(
    val id: String,
    val email: String,
    val age: UInt? = null,
    val tags: List<String>,
)
```

## Usage

Used internally by `typewriter-macros`. Most users should depend on the main [`typewriter`](https://crates.io/crates/typewriter) crate.
