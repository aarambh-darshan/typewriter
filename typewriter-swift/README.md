# typewriter-swift

> Swift emitter for the [typewriter](https://github.com/aarambh-darshan/typewriter) SDK.

[![Crates.io](https://img.shields.io/crates/v/typewriter-swift.svg)](https://crates.io/crates/typewriter-swift)
[![Docs.rs](https://docs.rs/typewriter-swift/badge.svg)](https://docs.rs/typewriter-swift)

## What It Generates

| Rust | Swift |
|---|---|
| `struct` | `struct Name: Codable` with `CodingKeys` |
| Simple `enum` | `enum Name: String, Codable` |
| Tagged `enum` | `enum Name: Codable` with associated values |
| `Option<T>` | `T?` |
| `Vec<T>` | `[T]` |
| `HashMap<K,V>` | `[K: V]` |

## Example Output

```swift
// Code generated by typewriter v0.3.0. DO NOT EDIT.
// Source: UserProfile

import Foundation

struct UserProfile: Codable {
    let id: UUID
    let email: String
    let age: UInt32?
    let tags: [String]

    enum CodingKeys: String, CodingKey {
        case id
        case email
        case age
        case tags
    }
}
```

## Usage

Used internally by `typewriter-macros`. Most users should depend on the main [`typewriter`](https://crates.io/crates/typewriter) crate.
