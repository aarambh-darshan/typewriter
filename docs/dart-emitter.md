# Dart/Flutter Emitter (Plugin)

The Dart emitter plugin generates `json_serializable`-compatible Dart classes for use with Flutter and other Dart projects.

## Installation

```bash
cd typewriter-plugin-dart
cargo build --release
cp target/release/libtypewriter_plugin_dart.so ~/.typewriter/plugins/
```

## Type Mappings

| Rust Type | Dart Type |
|-----------|-----------|
| `String` | `String` |
| `bool` | `bool` |
| `u8`–`u128`, `i8`–`i128` | `int` |
| `f32`, `f64` | `double` |
| `Option<T>` | `T?` |
| `Vec<T>` | `List<T>` |
| `HashMap<K, V>` | `Map<K, V>` |
| `(T1, T2, ...)` | `(T1, T2, ...)` (Dart 3.0 Records) |
| `Uuid` | `String` |
| `DateTime` | `DateTime` |
| `serde_json::Value` | `dynamic` |

## Usage

```rust
#[derive(TypeWriter)]
#[sync_to(dart)]
pub struct User {
    pub id: String,
    pub user_name: String,
    pub age: Option<u32>,
}
```

Generates `user.dart`:

```dart
// Auto-generated by typewriter v0.5.2. DO NOT EDIT.
// Source: User
// Regenerate: cargo typewriter generate

import 'package:json_annotation/json_annotation.dart';

part 'user.g.dart';

@JsonSerializable()
class User {
  final String id;
  @JsonKey(name: 'user_name')
  final String userName;
  final int? age;

  const User({
    required this.id,
    required this.userName,
    this.age,
  });

  factory User.fromJson(Map<String, dynamic> json) =>
      _$UserFromJson(json);
  Map<String, dynamic> toJson() => _$UserToJson(this);
}
```

> **Note:** You must run `dart run build_runner build` (or `flutter pub run build_runner build`) to generate the `*.g.dart` files required by `json_serializable`.

## Enums

### Unit Enums → Dart Enum

```rust
#[derive(TypeWriter)]
#[sync_to(dart)]
pub enum Role {
    Admin,
    User,
    Guest,
}
```

```dart
enum Role {
  @JsonValue('Admin')
  admin,
  @JsonValue('User')
  user,
  @JsonValue('Guest')
  guest;
}
```

### Complex Enums → Sealed Classes (Dart 3.0+)

```rust
#[derive(TypeWriter)]
#[sync_to(dart)]
pub enum Shape {
    Circle { radius: f64 },
    Rectangle { width: f64, height: f64 },
}
```

```dart
sealed class Shape {
  const Shape();
}

@JsonSerializable()
class Circle extends Shape {
  final double radius;

  const Circle({
    required this.radius,
  });

  factory Circle.fromJson(Map<String, dynamic> json) =>
      _$CircleFromJson(json);
  Map<String, dynamic> toJson() => _$CircleToJson(this);
}

@JsonSerializable()
class Rectangle extends Shape {
  final double width;
  final double height;

  const Rectangle({
    required this.width,
    required this.height,
  });

  factory Rectangle.fromJson(Map<String, dynamic> json) =>
      _$RectangleFromJson(json);
  Map<String, dynamic> toJson() => _$RectangleToJson(this);
}
```

## Flutter Integration

1. Add dependencies to your `pubspec.yaml`:

```yaml
dependencies:
  json_annotation: ^4.8.0

dev_dependencies:
  build_runner: ^2.4.0
  json_serializable: ^6.7.0
```

2. Generate your types:

```bash
typewriter generate --all --lang dart
```

3. Run build_runner:

```bash
dart run build_runner build
```

## Configuration

```toml
# typewriter.toml

[dart]
output_dir = "./generated/dart"
file_style = "snake_case"   # default
```
