# Ruby Emitter (Plugin)

The Ruby emitter plugin generates [Sorbet](https://sorbet.org/)-compatible `.rbi` type signature files from Rust structs and enums.

## Installation

The Ruby emitter is a plugin that must be built and installed separately:

```bash
cd typewriter-plugin-ruby
cargo build --release
cp target/release/libtypewriter_plugin_ruby.so ~/.typewriter/plugins/
```

## Type Mappings

| Rust Type | Ruby/Sorbet Type |
|-----------|------------------|
| `String` | `String` |
| `bool` | `T::Boolean` |
| `u8`–`u128`, `i8`–`i128` | `Integer` |
| `f32`, `f64` | `Float` |
| `Option<T>` | `T.nilable(T)` |
| `Vec<T>` | `T::Array[T]` |
| `HashMap<K, V>` | `T::Hash[K, V]` |
| `(T1, T2, ...)` | `[T1, T2, ...]` |
| `Uuid` | `String` |
| `DateTime` | `Time` |
| `NaiveDate` | `Date` |
| `serde_json::Value` | `T.untyped` |

## Usage

```rust
#[derive(TypeWriter)]
#[sync_to(ruby)]
pub struct User {
    pub id: String,
    pub name: String,
    pub age: Option<u32>,
}
```

Generates `user.rbi`:

```ruby
# typed: strict
# Auto-generated by typewriter v0.5.2. DO NOT EDIT.
# Source: User
# Regenerate: cargo typewriter generate

class User < T::Struct
  const :id, String
  const :name, String
  const :age, T.nilable(Integer), default: nil
end
```

## Enums

### Unit Enums → `T::Enum`

```rust
#[derive(TypeWriter)]
#[sync_to(ruby)]
pub enum Role {
    Admin,
    User,
    Guest,
}
```

```ruby
class Role < T::Enum
  enums do
    Admin = new('Admin')
    User = new('User')
    Guest = new('Guest')
  end
end
```

### Complex Enums → Sealed Modules

```rust
#[derive(TypeWriter)]
#[sync_to(ruby)]
pub enum Shape {
    Circle { radius: f64 },
    Rectangle { width: f64, height: f64 },
}
```

```ruby
module Shape
  extend T::Helpers
  sealed!

  class Circle < T::Struct
    include Shape
    const :radius, Float
  end

  class Rectangle < T::Struct
    include Shape
    const :width, Float
    const :height, Float
  end
end
```

## Configuration

```toml
# typewriter.toml

[ruby]
output_dir = "./generated/ruby"
file_style = "snake_case"   # default
```
