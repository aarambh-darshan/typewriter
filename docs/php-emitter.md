# PHP Emitter (Plugin)

The PHP emitter plugin generates PHP 8.1+ `readonly` classes with constructor-promoted properties from Rust structs and enums.

## Installation

```bash
cd typewriter-plugin-php
cargo build --release
cp target/release/libtypewriter_plugin_php.so ~/.typewriter/plugins/
```

## Type Mappings

| Rust Type | PHP Type |
|-----------|----------|
| `String` | `string` |
| `bool` | `bool` |
| `u8`–`u128`, `i8`–`i128` | `int` |
| `f32`, `f64` | `float` |
| `Option<T>` | `?T` |
| `Vec<T>` | `array` |
| `HashMap<K, V>` | `array` |
| `(T1, T2, ...)` | `array` |
| `Uuid` | `string` |
| `DateTime` | `\DateTimeInterface` |
| `serde_json::Value` | `mixed` |

## Usage

```rust
#[derive(TypeWriter)]
#[sync_to(php)]
pub struct User {
    pub id: String,
    pub name: String,
    pub age: Option<u32>,
}
```

Generates `User.php`:

```php
<?php
// Auto-generated by typewriter v0.5.2. DO NOT EDIT.
// Source: User
// Regenerate: cargo typewriter generate

declare(strict_types=1);

readonly class User
{
    public function __construct(
        public string $id,
        public string $name,
        public ?int $age = null,
    ) {}
}
```

## Enums

### Unit Enums → Backed Enum

```rust
#[derive(TypeWriter)]
#[sync_to(php)]
pub enum Role {
    Admin,
    User,
    Guest,
}
```

```php
enum Role: string
{
    case Admin = 'Admin';
    case User = 'User';
    case Guest = 'Guest';
}
```

### Complex Enums → Interface + Classes

```rust
#[derive(TypeWriter)]
#[sync_to(php)]
pub enum Shape {
    Circle { radius: f64 },
    Rectangle { width: f64, height: f64 },
}
```

```php
interface Shape {}

readonly class Circle implements Shape
{
    public function __construct(
        public float $radius,
    ) {}
}

readonly class Rectangle implements Shape
{
    public function __construct(
        public float $width,
        public float $height,
    ) {}
}
```

## Configuration

```toml
# typewriter.toml

[php]
output_dir = "./generated/php"
file_style = "PascalCase"   # default
```
